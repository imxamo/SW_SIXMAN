1학기 목표 설립

딥러닝 파트 (새로운 딥러닝 아이디어 필요)
	작물/질병 가지 수 늘리기 (작물 8개, 질병 16개)
	드라이브를 사용하는 코드를 완성한다면 여러명이 작업할 수 있음

하드웨어 (스마트팜 제작)
	1. 카메라 작동
	2. 서버랑 연결해서 DB에 데이터 전송
	3. 미니 케이스

서버 (Putty 서버 사용)
	1. 웹(이미지 입력) -> DB -> 모델 작동 -> DB 저장
	2. 웹(정보 탭 접속) -> DB -> 웹(결과 출력)
	3. 하드웨어(이미지 입력) -> DB -> 모델 작동 -> DB 저장
	4. 웹사이트 기본 골자


* SW중심대학에서 제공하는 Putty 서버 사용 *
  Putty서버와 깃허브를 연동하는 방식으로 DB, html 등 구현 내용을 깃허브에 업로드하여 사용함
  서버 포트번호를 부여받은 후 서버 내 작업공간에 꼭 mysql이 아니라도 다른 DB와 리액트 등을 설치하여 사용 가능할 것으로 보임.
  기존 스프링부트 방식으로 개발 진행하면 될 듯함
